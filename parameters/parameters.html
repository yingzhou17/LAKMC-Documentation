Parameters¶

The documentation for the parameters in the input file.

Units¶

Unless otherwise mentioned all units are as follows:

  • Time -> fs (femtosecond)
  • Distance -> Å (Angstroms)
  • Energy -> eV (electronvolt)
  • Force -> eV/Å (electronvolt per Angstrom)
  • Temperature -> K (Kelvin)

To be more precise; the default time, distance, energy and force units are those of the MD library you are using. When using LBOMD the default units will be as above. If you are using LAMMPS with metal units then they will be the same too.
