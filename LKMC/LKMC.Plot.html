LKMC.Plot.linePlot(filename, xvals, xlabel, yvals, ylabel, title=None, label=None, grid=1, textsize=20, useInterp=1, yerror=None, xlimmin=None, xlimmax=None, ylimmin=None, ylimmax=None, ymax=None, tight=True)[source]¶

Do a line plot of data using stineman interpolation. ‘ymax’ should be specified as the index in the arrays of the maximum

(so that interpolation doesn’t change it).

‘label’ goes in the legend if you specify it. Saves graph in ‘filename’.pdf.

LKMC.Plot.multiLinePlot(filename, xvalsList, xlabel, yvalsList, ylabel, labels, ymax=0, ymin=0, title=None, grid=1, textsize=20, colours=None, useInterp=1)[source]¶

Do a line plot of multiple sets of data. Data is interpolated using Stineman interpolation ‘xvalsList’: list of arrays containing xvals for each set of data ‘yvalsList’: corresponding list of arrays containing y values ‘labels’: list containing a label for each set of data ‘ymax’: boolean, if true will interpolate up to and down

from the max value

‘ymin’: boolean, equivalent to ymax Saves graph in ‘filename’.pdf.
