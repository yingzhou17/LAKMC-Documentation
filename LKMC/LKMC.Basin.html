LKMC.Basin module¶

This module is an implementation of the Superbasin Method, which deals with the low-barrier flicker problem.

Copyright Miao Yu 2014 Copyright Mark James Wootton & Adam Lloyd 2016

References¶

  • L.K. Beland, P. Brommer, F. El-Mellouhi, J.-F. Joly and N. Mousseau. Kinetic activation-relaxation techinique. Phy. Rev. E 84, 046704 (2011) doi: 10.1103/PhysRevE.84.046704
  • B. Puchala, M.L. Falk and K. Garikipati. An energy basin finding algorithm for kinetic Monte Carlo acceleration. J. Chem. Phys. 132, 134104 (2010) doi: 10.1063/1.3369627
  • K.A. Fichthorn and Y. Lin. A local superbasin kinetic Monte Carlo method. J. Chem. Phys. 138, 164104 (2013) doi: 10.1063/1.4801869

Note: The script is coded based on Beland’s work.

class LKMC.Basin.BasinObject(KMCRunner)[source]¶

Bases: object

The BasinObject class holds information about a superbasin.

Parameters:

KMCRunner : KMC.KMCRunner object

We need some information/methods from this object.

Attributes

defectVolumes (list) List of DefectVolumeObject objects, holds information about all basin states in this superbasin.
connectivity (list (2D matrix)) Hold information about the connectivity within this superbasin.
params (Input.InputParams object) This object contains all parameters which are read from lkmcInput.IN by default.
cellDims (ndarray) Cell dimensions of the system. Used for PBC.
pointer (list of integer) Store the start/end point for each basin state. e.g. For a superbasin with three basin states, which, respectively, have 3,4,5 transitions. The pointer will be [0,3,7,12].
currentDV (integer) The index of basin state where the system currently is.
currentTrans (integer) The index of transition which has been chosen in the last step.
previousDV (integer) The index of basin state where the system is on previous step. This is a backup for previous currentDV. currentDV will fall back to this value if we failed to perform chosen transitions in the superbasin in current step.

shortestPath(start, end)[source]¶

Find the shortest pathway from the starting basin state to the ending basin state by using the Dijkstra’s algorithm.

Parameters:

start : integer

Index of the starting basin state in self.defectVolumes.

end : integer

Index of the ending basin state in self.defectVolumes.
Returns:

S : list

The pathway from start to end. Notes :

  • If return empty list : there’s no path from start to end;
  • If the start == end : return a list with a single point;
  • If start == -1 : start = last DV.
