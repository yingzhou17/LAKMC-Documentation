class LKMC.Writer.WriterItem(job, dataType=None, data=[], outdir=None, filename=None)[source]¶

Bases: object

An object to be placed into LKMCWriter queue.

Parameters:

job : str

The name of the job. Does nothing currently. If you set job to ‘EXIT’ the writer process will terminate.

dataType : {‘plot’, ‘object’}

The type of data you are passing. If you are writing an object dataType should be ‘object’, whereas if you are doing a plot then dataType should be ‘plot’.

data : list, optional

For ‘object’ dataType the first item in the list should be the object. For ‘plot’ dataType this list is not required.

outdir : str

The directory to write into. Relative paths should be specified relative to the params.outputDirectory.

filename : str, optional

The name of the file to write. Generally not required as it is normally part of the args tuple.

Attributes

commands (list) List of commands to run on the object or plots to plot. Each item should be a tuple containing: * the objects method you want to run / plot you want to plot (str) * the tuple containing the positional arguments for the method or plot (tuple) * the dictionary of key word arguments for the method or plot (dict)
