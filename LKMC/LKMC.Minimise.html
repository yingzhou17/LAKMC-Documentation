calcForce(obj, excl, params, includeAtoms=None, mainIndexes=None)[source]¶

Force calculation.

Parameters:

obj :

The object being minimised.

excl : {0, 1, 2}

The defect flag to set when calling calcForce on obj.

params : Input.InputParams

The parameters instance.

mainIndexes : ndarray, optional

An array containing the indexes of the atoms that we need to accurately determine the forces for.

includeAtoms : ndarray, optional

An array containing the indexes of the atoms to include during the force calculation (for a sub-lattice force calculation). The forces on these atoms will not be correct but including them ensures the forces on the mainIndexes atoms are.
Returns:

status : integer

0 mean success, anything else means a failure.

class LKMC.Minimise.LBFGSMinimiser(params, noFallback=False)[source]¶

Bases: LKMC.Minimise.BaseMinimiser

Limited memory BFGS minimiser from SciPy.

Parameters:

params : Input.InputParams

The parameters object.

noFallBack : bool, optional

If ‘True’ do not run fall back minimisations.

References

  • C. Zhu, R. H. Byrd and J. Nocedal. L-BFGS-B: Algorithm 778: L-BFGS-B, FORTRAN routines for large scale bound constrained optimization (1997), ACM Transactions on Mathematical Software, 23, 4, pp. 550 - 560.
  • http://docs.scipy.org/doc/scipy/reference/generated/scipy.optimize.fmin_l_bfgs_b.html

Attributes

rVector (ndarray) The final eigenvector array, corresponding to the lowest eigenvalue.
mainIndexes (ndarray) The indexes of the atoms that have been used in a sub-lattice calculation.
includeAtoms (ndarray) The indexes of the additional atoms that have been used in a sub-lattice force calculation.
