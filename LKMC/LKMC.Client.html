LKMC.Client module¶

LKMC Client script.

Runs as daemon on remote machine and connects to server process. Spawns worker processes to do the jobs.

LKMC.Client.main()[source]¶

The main client loop. First we create a queue manager which connects to the queues on the server. We then create “NWorkers” worker processes and pass them the queues. We then wait for the workers to exit, and then return. (We should probably listen on a special queue from the server so that it can tell us to exit, eg if received sigint...)
