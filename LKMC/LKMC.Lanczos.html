LKMC.Lanczos.calcHessianVectProduct(lattice, mainIndexes, includeAtoms, displVectorNormalised, parameters)[source]¶
Method approximates a dot product between a Hessian matrix and a vector of a
given system. lattice: input lattice mainIndexes: list of atoms that should be included in Hessian matrix calculation displVectorNormalised: a random normalised vector parameters: input parameters from lkmcInput.IN
Returns:
hessianU: a dot product of a hessian matrix and a normalized vector of a
given system.

forEvalCnt: number of force evaluations used to calculate hessian matrix

REMARKS: be sure that the passed lattice has an evaluated force vector
