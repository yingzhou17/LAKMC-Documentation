LKMC.updateParameters module¶

This module allow user to update/compare their input parameter file.

@author Miao Yu 2015

LKMC.updateParameters.commandLineArgs()[source]¶

Parse command line arguments

LKMC.updateParameters.main()[source]¶
LKMC.updateParameters.run(inputFile_new, inputFile_old, print_flag=False, compare_flag=True, return_flag=True)[source]¶

This script will merge old input file into new input file. Params not in new will be ignored, params not in old will be added, param values equal to old if exist, all comments will be from new.
