extractData(lattice)[source]¶

Get data back from LAMMPS and store it.

Parameters:

lattice : Lattice.Lattice

The Lattice object that the LAMMPS data will be stored in.

Notes

We assume that lattice is the Lattice that was used to run LAMMPS. If the number of atoms in LAMMPS differs to the number in lattice we remove extra atoms in lattice.

minimise(lattice, initialise=True)[source]¶

Run a minimisation using LAMMPS.

Parameters:

lattice : Lattice.Lattice

The lattice object to be minimised.

initialise : True or False, optional

If True (default) then initialise LAMMPS before running the minimisation. If the lattice has not been modified since the last time LAMMPS was called then you do not need to initialise.
