LKMC.Images module¶

Contains the Images class and associated methods.

@author: Chris Scott

class LKMC.Images.Images(params, numProcs=1)[source]¶

Bases: object

The Images class contains a number of images (or states) of a system.

Parameters:

params : Input.InputParams

The input parameters object.

numProcs : int, optional

The number of processors to use for the force calculation. Defaults to 1.

Attributes

NAtoms (int) The total number of atoms in all the (moving) images.
imageNAtoms (int) The number of atoms in a single image.
pos (ndarray) The positions of all the atoms from all the images.
fixedEndPoints (bool) Whether the end points of the band are fixed or not.
interpolationMethod (str) The method that was used to interpolate the band.

interpolateImages(initialState, finalState, NImages, method='linear', saddleState=None, fixedEndPoints=True, stepSize=None)[source]¶

Interpolate a number of images between initial and final states.

Parameters:

initialState : Lattice.Lattice

The initial state for the band of images.

finalState : Lattice.Lattice

The final state for the band of images.

NImages : int

The number of images to create in total, including the initial and final.

method : str, optional

The method to use for the interpolation. Currently only support “linear” (“cubicspl” should be added at some point).

saddleState : Lattice.Lattice, optional

If this argument is provided we interpolate from the initial to here and then from here to the final.

fixedEndPoints : bool, optional

If True then the end points (initial and final) of the band are fixed (i.e. they cannot move). Defaults to True.

stepSize : float, optional

If fixedEndPoints is False and this kwarg is set then we step either side of the saddle (towards initial/final) by this amount and use those points as the end points.

writeBarrierSeparation(filename, initialEnergy=None, initialPos=None)[source]¶

Write csv file containing details of the images.

Parameters:

filename : str

The name of the file to create.

initialEnergy : float, optional

If set this is used for the relative energy calculation.

initialPos : ndarray, optional

If set we use this array for the separation calculation.

Notes

If the end points are not fixed it is important to either set the Images.initialEnergy attribute or the initialEnergy kwarg so that the relative energies are calculated correctly.

Image separation is the piecewise linear separation from the initial image. If the Images.initialPos attribute or the initialPos kwarg is set we use that for the separation calculation.

The file will contain the image number, the piecewise linear separation of the image from the initial image, the total energy of the image and the relative energy of the image with respect to the initial energy.
