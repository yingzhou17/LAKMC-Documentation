class LKMC.Defects.DefectVolumesFinder(params)[source]¶

Bases: object

Find defects and construct defect volumes.

Parameters:

params : Input.InputParams

The input parameters object.

Notes

The workflow is to first identify defects and then to construct the defect volumes. The method for identifying defects is set in the parameter file, for example it can be by comparison to a reference lattice or by looking for undercoordinated atoms.
